# Estoque Atual Fitness

Aplicativo web instalavel para controle local de estoque da Atual Fitness.

## Funcionalidades

- Cadastro de produtos
- Geracao de SKU interno
- Controle de entrada, saida e ajuste
- Dashboard de estoque
- Historico de movimentacoes
- Funcionamento offline por dispositivo

GitHub Pages:

## Observacao

Esta versao usa armazenamento local do navegador. Para sincronizar dados entre varios aparelhos, e necessario backend e banco de dados online.
